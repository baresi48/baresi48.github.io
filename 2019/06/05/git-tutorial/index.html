1 客户端安装

1.1 Windows

  1. TortoiseGit：大部分人都用过TortoiseSVN，同样它也有Git版，仅支持 Windows 系统
  2. SourceTree：方便易用的客户端，同时支持 Windows 和 Mac 操作系统，Bitbucket官方推荐，除占用资源相对较高外几乎没什么缺点
  3. 例如VScode等很多IDE都带有自集成的GIT功能，稍加配置后即可使用

1.2 Linux

命令行才是 *nix 的灵魂！

apt install git

1.3 Mac OS

仅供参考：git-osx-installer

2 开始使用

考虑到用 *nix 系统的都是高手，应该不需要额外指导。故以下均以 Windows 系统 + TortoiseGit 为例进行说明。

2.1 下载并安装 Git

推荐：Git for Windows

2.2 安装 TortoiseGit

略。（官网上有汉化包，是否安装看个人喜好。）

2.3 配置密钥

在windows开始菜单中，找到刚才安装的TortoiseGit目录，其中有名为 PuttyGen 的工具。

  1. 点击运行
  2. 点击“Generate”按钮，鼠标在进度条位置==来回移动==直到进度条完毕，就会自动生一个随机的key
  3. 将“Key Comment”中内容改为自己姓名的拼音全拼
  4. 点击“Save public key”，保存为文件，并发送给Git管理员
  5. 点击“Save private key”，保存为文件，自行保存后备用

2.4 取出Git库

  1. 资源管理器，在准备放置Git库目录处，鼠标右击，菜单中选择 “Git 克隆”，在跳出的界面中按如下内容填写
  2. “URL” ： git@172.16.1.248:/home/git/source.git
  3. 选中“加载Putty密钥”：选择刚才PuttyGen生成的私钥文件
  4. 点击确定，取出远程的Git库
